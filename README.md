# 🐳 Docker Commands

### Build Docker Image

```bash
docker build -t cloud-resume-portfolio .
```

### Run Container

```bash
docker run -d -p 80:80 cloud-resume-portfolio
```

### Check Containers

```bash
docker ps
```

# 💻 Local Development

Clone the Repository:

```bash
git clone https://github.com/aibyp/cloud-resume-portfolio.git
```

Install Dependencies:

```bash
npm install
```

Run Locally:

```bash
npm run dev
```

Build Project:

```bash
npm run build
```

Preview Build:

```bash
npm run preview
```
